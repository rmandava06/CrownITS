# CrownIT Static React App

Static React recreation of the core public pages from [crownits.com](https://crownits.com/).

## Included Routes

- `/` Home page content and service highlights
- `/company` Company overview
- `/services` Services page
- `/careers` Careers contact page
- `/lca-eta-9035` Public LCA ETA 9035 links
- `/contact-us` Contact page
- `/terms-and-conditions` Terms and conditions summary
- `/blog/hello-crownit` Blog post clone

## Run Locally

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```
